# Bazic

An interpreted programming language written in Go

### How to use

Compile

```$ go build github.com/cedricmar/bazic/cmd/bazic```

Running a program

```$ ./bazic file.bz```

Run from prompt

```$ ./bazic```

## Tooling

### Generating AST

Use to generate expressions types

```$ go build github.com/cedricmar/bazic/cmd/generate_ast```

Then run

```$ ./generate_ast```
